4.Swaping two numbers:

	Neha
	pseudo: 
    		Input num1
		Input num2
 		num1 = num1 + num2
		num2 = num1 - num2
		num1 = num1 - num2 
		Return (num1,num2)

	Gopika 
	pseudo: routine 
    		Function SwapTwoNumbers(num1,num2)
		num1 = num1 + num2
		num2 = num1 - num2
		num1 = num1 - num2 
        	Return (num1,num2)
    
	Shilpa

    		Procedure TestSwapTwoNumbers()

		Output num2
